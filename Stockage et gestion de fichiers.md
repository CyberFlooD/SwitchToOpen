## 🔐 Outils open source pour le partage sécurisé de fichiers <a id="outils-partage-securise"></a>

| 🌟 **Outil** | 🔑 **Type de licence** | 🚀 **Fonctionnalités clés** | 📝 **Description** | 🌐 **Lien Web** |
|---|---|---|---|---|

| **FileRun** | Open Source | Gestion de fichiers, partage sécurisé | Gestionnaire de fichiers et plateforme de partage de fichiers en ligne open source avec une interface conviviale, qui supporte également le partage sécurisé | <div align="center"><a href="https://filerun.com/">🔗</a></div> |
| **Nextcloud** | Open Source | Stockage, partage sécurisé | Solution de cloud personnel open source qui permet le stockage et le partage de fichiers de manière sécurisée, avec des options de collaboration | <div align="center"><a href="https://nextcloud.com/">🔗</a></div> |

| **Pydio Cells** | Open Source | Partage de fichiers, collaboration | Plateforme de partage de fichiers et de collaboration open source, offrant un contrôle total sur le stockage et la sécurité des données | <div align="center"><a href="https://pydio.com/">🔗</a></div> |
| **OnionShare** | Open Source | Partage anonyme, basé sur Tor | Application de partage de fichiers open source, qui utilise le réseau Tor pour partager des fichiers de manière anonyme et sécurisée | <div align="center"><a href="https://onionshare.org/">🔗</a></div> |
| **CryptPad** | Open Source | Suite collaborative, partage chiffré | Suite collaborative open source avec un outil de partage de fichiers chiffré, permettant aussi l'édition de documents de manière sécurisée | <div align="center"><a href="https://cryptpad.fr/">🔗</a></div> |
| **SparkleShare** | Open Source | Partage basé sur Git, collaboration | Outil de partage de fichiers open source conçu pour les équipes, basé sur Git, idéal pour les fichiers de projet et la collaboration | <div align="center"><a href="https://sparkleshare.org/">🔗</a></div> |
| **Plik** | Open Source | Partage sécurisé, temporaire | Application de partage de fichiers auto-hébergée, permettant des transferts sécurisés et un stockage temporaire des fichiers | <div align="center"><a href="https://github.com/root-gg/plik">🔗</a></div> |
